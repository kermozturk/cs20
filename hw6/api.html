DOG API


About the API


The API I selected retrieves random images of dogs. The main reason I used this API was I lost a lot of time finding the appropriate API to use. Initially, I tried using Google Flight API and retrieving flight data from the API. However, even though my get request was successfuk, it returned an error message. It turned out that the API was problematic, but it costed me a lot of time. So, I wanted to go with a simple API that gives dog images. Also, I really like dogs and seeing them on my screen made me really happy.



Website: https://dog.ceo/dog-api/documentation/



Parameters Used: The parameter used is the breed name in the URL in my url, doberman is my parameter, so the API only returns images of dogs that have the breed doberman



This API could be helpful if someone decided to learn more about different dog breeds and wanted to see what they look like. For example, it could be used in a website that quizzes the users on dog breeds. The answers could be compared to the breed parameter.
